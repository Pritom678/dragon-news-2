Dragon News is a **modern, newspaper-style web application** built with **React**, **React Router**, **HTML**, **CSS**, and **Tailwind CSS**. It provides users with the latest news across multiple categories including **sports, entertainment, politics, international news, games**, and more. The app is fully **responsive**, **interactive**, and beginner-friendly.  

---

## Features

- **Local JSON Data Source:** All news articles are stored locally in a JSON file for easy management and faster loading.  
- **Category-wise News:** Explore news by categories like Sports, Entertainment, Politics, Games, and more.  
- **Dynamic Routes:** Each news article has its own page with a unique URL for easy navigation and sharing.  
- **Protected Routes:** Certain pages/features are accessible only under specific conditions for smooth user experience.  
- **Bookmark Functionality:** Save favorite articles for quick access later.  
- **Responsive UI:** Works perfectly on mobile, tablet, and desktop screens.  
- **Newspaper-style Layout:** Clean, classic design for easy reading and navigation.  

---

## Tech Stack

- **Frontend:** React, React Router, HTML5, CSS3, Tailwind CSS  
- **Data Source:** Local JSON files  

---

## How it Works

1. News articles are fetched from a **local JSON file**.  
2. Users navigate through different **categories** using the main menu.  
3. Clicking on an article opens a **dynamic route** showing full details.  
4. Users can **bookmark** articles to save them locally for later reading.  
5. The UI adjusts seamlessly across all device sizes for a smooth reading experience.
